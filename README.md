# Dynamic Menu System (C++)

## Description
Console-based application developed in C++ that integrates multiple functionalities through a dynamic menu system. The program allows users to navigate between different modules and perform various operations.

## Features
- Interactive menu navigation using loops and conditionals
- Modular program design using functions
- Multiple integrated systems within one program:
  - International System of Units (SI)
  - Numerical base conversions (decimal, binary, octal, hexadecimal)
  - Dice simulation with statistical analysis (mean, median, mode)
  - Magic square generation and validation

## Technical Implementation
- Structured programming with functions
- Use of arrays and matrices
- Input validation for user data
- Random number generation for simulations
- Mathematical calculations and algorithms

## Methods Used
- Numerical base conversion algorithms
- Statistical calculations (mean, median, mode)
- Magic square generation using the diagonal exchange method
- Validation of results through mathematical verification

## Technologies
- C++
- Standard libraries (iostream, cmath, cstdlib, ctime, iomanip)

## Purpose
Academic project focused on:
- Applying programming logic
- Designing modular systems
- Implementing algorithms and mathematical concepts
- User interaction via console
